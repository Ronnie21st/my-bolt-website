📦 INSTALLATION STEPS:

1. CREATE ZIP FILE (if not already zipped):
   - Compress the "bolt-wp-theme" folder
   - Make sure the folder structure is preserved
   - OR use the provided .tar.gz file

2. UPLOAD TO WORDPRESS:
   a. Log into your WordPress Admin Dashboard
   b. Navigate to: Appearance → Themes
   c. Click "Add New" button
   d. Click "Upload Theme" button
   e. Click "Choose File" and select your ZIP
   f. Click "Install Now"
   g. Wait for installation to complete
   h. Click "Activate"

3. DONE! 🎉
   - Theme is now active
   - All animations working
   - Fully responsive
   - Ready to customize

🎨 USING ELEMENTOR (OPTIONAL):

1. Install Elementor Plugin:
   - Plugins → Add New
   - Search "Elementor"
   - Install & Activate

2. Edit with Elementor:
   - Create/Edit any page
   - Click "Edit with Elementor"
   - Drag and drop widgets
   - All theme styles apply automatically

3. Choose Template:
   - Page Attributes → Template
   - Select: Default, Elementor Full Width, or Canvas

🆘 TROUBLESHOOTING:

Issue: Theme won't install
→ Solution: Ensure ZIP contains "bolt-wp-theme" folder at root

Issue: Styles not loading
→ Solution: Clear browser cache and WordPress cache

Issue: Mobile menu not working
→ Solution: Check JavaScript is enabled in browser

Issue: Elementor compatibility
→ Solution: Update Elementor to latest version
